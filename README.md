It started with lightclient. Teal squares on white — simple, clean, just watching the chain do its thing. That was supposed to be it.
Then darkclient showed up. Same squares, same tower, but flipped — glowing against the void. And once there were two, well, the rest was inevitable. 
The colors just kept coming. 300 of them, each one placed on a golden-angle spiral so no two neighbors look alike. Some are loud, some are quiet, some you have to squint at. 
All of them are the Tower.
